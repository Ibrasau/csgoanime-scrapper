# csgoanime-scrapper
### WARNING: SCRAPPED CONTENT MIGHT BE NSFW!
#### Usage:
```bash
git clone git@github.com:Ibrasau/csgoanime-scrapper.git
cd csgoanime-scrapper
node ./scrapper.js 
```
Scrapped *.webm files will appear in ./scrapped directory.
